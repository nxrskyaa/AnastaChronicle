# Anasta Chronicle

**Pixel art 2.5D** top-down forest adventure.

## Play
https://anasta-chronicle.vercel.app

## Features
- 100% code-generated pixel world, characters, creatures, props, and boss art
- Depth-sorted 2.5D draw order
- Class combat · nearest-target ranged aim · layered weapon/skill FX
- Shared Infernyx world boss · synchronized attacks · equal co-op rewards
- Mutual opt-in player duels · reconnecting realm chat · speech bubbles
- Tension fishing · illustrated Fish Chronicle · rod progression · guided forge
- 28 active NPCs across six regions with day/night routines
- Character creator · 14-pet sanctuary · switchable active companion
- Leveling · quests · inventory · minimap · day/night and weather
- Touch joystick for mobile

## Controls
WASD · LMB attack · Shift shield · Space evade · F interact/fish · C forge · I bag · Q quests · J Fish Chronicle · P next pet · Enter chat · 1–4 skills

python3 -m http.server 5173
